# Target Alert System

A Python-based automation project that reads designer targets from Google Sheets and sends personalized weekly email alerts.  
This system helps track performance and notify designers about completed, pending, and upcoming tasks using AI-powered messages.

---


## Project Overview

This project automates the process of notifying designers about their weekly targets by:

- Reading data from a Google Sheet
- Calculating completed and pending targets
- Generating dynamic AI-based encouragement messages
- Sending professional HTML email alerts to each designer(In Progress)

This helps **increase productivity**, reduce manual tracking, and provide **real-time updates** for team members.

---

## Features

- ✅ Google Sheets integration for real-time target tracking
- ✅ Personalized AI-generated messages based on target completion
- ✅ Professional HTML email template
- ✅ Email automation using Python and Gmail SMTP
- ✅ Dynamic table with targets, completed work, and pending tasks
- ✅ Supports multiple designers in a single sheet

---

## Tech Stack

- **Python 3.10+**
- **gspread** – for Google Sheets API
- **oauth2client** – for Google authentication
- **smtplib & email.mime** – for sending emails
- **HTML/CSS** – for professional email formatting
